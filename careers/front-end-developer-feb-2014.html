Front End Developer

Role And Benefits

  • Take part in web/mobile creative applications on HTML5 platform an with most updated web front-end techniques.
  • To be provided with utmost supportive facilities and policies to develop technical skills and to research on new technologies and new trends.
  • Chances to work with harmonious and united team with the same passion for innovation
  • Receive deserving salary and benefits.

Qualities

  • Good observation skill and a sharp eye for every design detail
  • Willing to learn, and quick at catching up
  • Passionate about technologies and sharing knowledge with others
  • Willing to listen to co-workers' feedbacks and actively research for solutions that improve effectiveness.
  • Good English communication skills

Skills

  • Fluent in creating websites using HTML + CSS
  • Experienced in using JavaScript and jQuery
  • Know how to use HTML5, CSS3
  • Experienced in Responsive Web Design
  • Experienced in manual conversion from PSD (Photoshop) to HTML+CSS
  • Experienced in using version control systems such as Git, SVN

Bonus Points

  • No or very little experience in using DreamWeaver to create websites
  • Experienced in interacting with back-end via web service / RESTful / AJAX
  • Experienced in using one of the JavaScript frameworks (Backbone, Angular, Ember…)
  • Knowledgeable of page load speed optimization, JavaScript and jQuery optimization
  • Knowledgeable of Semantic Web và Progressive Enhancement
