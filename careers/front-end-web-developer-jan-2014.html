Front End Web Developer

Nâu Studio đang cần tuyển Front-end Web Developer (Lập trình viên giao diện web). Bạn sẽ có cơ hội làm việc với những người bạn hoà đồng, ăn ý, luôn thích thú sáng tạo ra các ứng dụng web mới mẻ.

Bạn là người:

  • Có khả năng quan sát và nhạy bén về giao diện, sự tương tác, chuyển động
  • Ham học hỏi và tiếp thu nhanh
  • Có đam mê và thích chia sẻ về các công nghệ web đang và sắp thịnh hành
  • Không thích làm những việc lặp lại và luôn tìm cách tự động hoá những việc nhàm chán
  • Quan niệm viết code là cho người khác đọc, code dễ hiểu và ghi chú thật nhiều
  • Biết tiếp thu ý kiến từ đồng nghiệp và tích cực đóng góp để cải thiện hiệu suất công việc

Bạn phải có những kỹ năng sau:

  • Vững HTML5, CSS3
  • Vững jQuery, nhưng khi cần có thể viết bằng JavaScript không
  • Có kinh nghiệm về Thiết Kế Web Linh Động (Responsive Web Design)
  • Có kinh nghiệm chuyển đổi thiết kế trên Photoshop / Illustrator sang HTML + CSS
  • Có kinh nghiệm với một trong các ngôn ngữ back-end PHP/.NET/Java/Ruby/Python để tạo trang web động
    HOẶC có kinh nghiệm tương tác với back-end thông qua web service / RESTful / AJAX.
  • Có kinh nghiệm sử dụng hệ thống quản lý phiên bản (version control system) như Git, SVN

Nếu bạn có những kỹ năng sau thì tuyệt vời:

  • Có kinh nghiệm viết web app trên di động
  • Có kinh nghiệm sử dụng một trong các ngôn ngữ CSS mở rộng như SASS hoặc LESS
  • Có kinh nghiệm sử dụng một trong các framework như Backbone, Angular, hoặc Ember
  • Có kinh nghiệm với Node.js và công cụ tự động hoá Grunt.js
  • Có kinh nghiệm viết JavaScript dễ quản lý và mở rộng, biết sử dụng JavaScript pattern, closure, AMD
  • Hiểu biết về tối ưu hoá tốc độ tải website, tối ưu hoá jQuery và JavaScript, biết sử dụng devtools để debug và tìm điểm yếu về tốc độ thực thi.
  • Hiểu biết về Web Có Nghĩa (Semantic Web)
  • Hiểu biết về Progressive Enhancement (nói chung là làm cho website chạy được từ IE cho đến Chrome)

Bạn hãy yên tâm vì Nâu không coi trọng số năm kinh nghiệm của bạn. Chúng tôi sẽ chỉ đánh giá bạn về kỹ thuật, những kinh nghiệm thực tiễn và những dự án đã tham gia, cùng khả năng hoà nhập tốt với nhóm trong ngày thử việc.
